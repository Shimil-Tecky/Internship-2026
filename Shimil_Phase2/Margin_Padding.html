Margin vs Padding Examples

Example 1: Padding Only

This box has 50px padding (space inside the border, around the text).

Padding adds space INSIDE the element, between content and border.

Example 2: Margin Only

This box has 50px margin (space outside the border, between other elements).

Margin adds space OUTSIDE the element, between this box and other elements.

Example 3: Both Margin and Padding

This box has 50px margin AND 50px padding combined.

Margin creates external spacing, while padding creates internal spacing.
